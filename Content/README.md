# Content Directory

This directory contains the AI components for the Assaultron project.

## Ollama
- Download and place Ollama executable in the `Ollama` folder
- Recommended models: llama3.2:latest, mistral, or codellama

## xVAsynth
- Download xVAsynth from: https://github.com/DanRuta/xVA-Synth
- Extract to the `xVAsynth` folder
- Voice models will be added later for Assaultron-specific speech

## Setup Instructions

### Ollama Setup:
1. Download Ollama from https://ollama.ai/
2. Install and run: `ollama serve`
3. Pull a model: `ollama pull llama3.2:latest`

### xVAsynth Setup (Future):
1. Download from GitHub releases
2. Extract to xVAsynth folder
3. Install Fallout voice models
